# Road_Labeler
Lane, Road Mark, Traffic Sign, Vanishing Point Labeling Tools
![DEMO2](/image/demo2.png)

## Features
* Support Lane, Road Mark, Traffic Sign, Vanishing Point Labeling Tools
* Dragable Points

## Dependency
* Python 3
* PyQt5
* Matplotlib
* Pillow
